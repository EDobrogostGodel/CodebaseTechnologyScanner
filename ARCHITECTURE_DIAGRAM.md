## Request/Response Example

```
User Action: Upload ZIP file
?????????????????????????????

1. Frontend Request:
   POST http://localhost:5000/api/scan/upload
   Content-Type: multipart/form-data
   Body: {
     file: [ZIP Binary Data],
     projectName: "MyProject"
   }

2. Backend Processing:
   ? ScanController receives request
   ? Extracts Stream from IFormFile
   ? Calls ScanService.ScanProjectAsync()
   ? Service extracts ZIP to temp folder
   ? TechnologyDetector analyzes files
   ? Creates ScanResult object
   ? Repository saves to JSON file
   ? Returns ScanResult

3. Backend Response:
   Status: 200 OK
   Content-Type: application/json
   Body: {
     "id": "abc-123-def-456",
     "timestamp": "2025-01-22T10:30:00Z",
     "projectName": "MyProject",
     "technologies": [
       { "name": "React", "type": "Framework", "version": "^18.2.0" },
       { "name": "TypeScript", "type": "Language", ... }
     ],
     "languageCounts": {
       "TypeScript": 15,
       "JavaScript": 3
     }
   }

4. Frontend Handling:
   ? scanService returns ScanResult
   ? Navigate to /results/abc-123-def-456
   ? ScanResultsPage loads and displays data
   ? ScanResultCard renders technologies
```
